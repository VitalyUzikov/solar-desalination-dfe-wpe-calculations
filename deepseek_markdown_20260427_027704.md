# Солнечная опреснительная установка DFE + WPE — открытая технология распределённого опреснения

**Ключевые параметры:**  
- 10-ступенчатый барабанный плёночный испаритель (DFE), барабаны 0,8×20 м  
- Производительность: 3,6 т/ч дистиллята, GOR = 8,8  
- Солнечная генерация пара (DSG): 410 кг/ч, 1,6 МПа, поле 900 м²  
- Эжекторная рециркуляция (μ=2,5), аккумулятор-сепаратор 6 м³  
- Пруд-испаритель с ветровым усилением (WPE): 3 барабана диаметром 2,0 м, сетка 2,5 м, ротор Савониуса 2,0×2,4 м  
- Zero Liquid Discharge (ZLD)
